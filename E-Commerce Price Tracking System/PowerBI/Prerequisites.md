# Prerequisites of connecting to PowerBI Online :

- You need to have a Azure Account.
- You need to have a tenant for logging into Microsoft Fabric.
  (Most of the Students are not able to configure the fabric because they are not having the Organization Account to login into Microsoft).
  So for this we can create a personal microsoft organization using the tenant properties in azure.If you face any issue while logging into Microsoft Fabric Feel Free to check on the vedio below to connect to it :

  Link : https://www.linkedin.com/posts/abhishek-srivastava-engineer_microsoftfabric-powerbi-dataanalytics-activity-7392932527142797312-75Xi?utm_source=share&utm_medium=member_desktop&rcm=ACoAAC8PtfsBQvOdoRI3GojKWN8wdZYsRiN66Vw


Steps to connect to PowerBI : 

- Open Azure.
- Login into Microsoft Fabric.
- Switch to PowerBI Service.
